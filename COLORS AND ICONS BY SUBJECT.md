# COLOR ACCENTS BY TOPIC
Splash requires certain primary color accents and provides a specific icon for some particular topics. These colors or icons should be used whenever a graphic covers one of these topics. If multiple topics are covered-- you are only required to use the icons. Where a project or set of posts that covers multiple topics all have a common color scheme that diverge from these -- use the appropriate color as an accent. Always follow the main brand color guide for any other colors. NEVER use colors other than those in these two sets.
